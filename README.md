# notes-app

This is a Vue.js based web application for taking notes.

It also has additional features such as favoriting notes and a specific page to show your favorite notes.

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
